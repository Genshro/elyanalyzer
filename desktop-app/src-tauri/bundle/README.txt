ElyAnalyzer Desktop v1.0.0
=============================

Installation Instructions:
1. Extract this ZIP file to a folder
2. Right-click on elyanalyzer-desktop.exe
3. Select 'Properties' > 'Unblock' if there's a security warning
4. Run elyanalyzer-desktop.exe
5. If Windows shows a security warning:
   - Click "More info"
   - Click "Run anyway"

Important Notes:
- This application is safe but not digitally signed yet
- Both elyanalyzer-desktop.exe and analysis-engine.exe must be in the same folder
- Requires Windows 10 or later
- Internet connection needed for login
